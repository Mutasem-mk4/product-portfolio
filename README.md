# Product Portfolio

Portfolio site for showcasing selected work, security tooling, and product case studies.

## Featured Project

### [procscope](https://github.com/Mutasem-mk4/procscope)

eBPF runtime investigation tool for Linux malware triage and incident response.

- Process-scoped tracing for exec, file, network, privilege, namespace, and mount activity
- Debian packaging and DEP-8 autopkgtests prepared upstream
- Submitted to ParrotSec for distro inclusion
- Submitted to BlackArch via upstream package PR
